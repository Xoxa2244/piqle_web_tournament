# Инструкция по импорту игроков

## ✅ Файл готов: `player_list_import_ready.csv`

### Статистика:
- **116 игроков** из **29 команд**
- **4 дивизиона**: DUPR 12, DUPR 14, DUPR 16, DUPR 18
- Формат: **4v4** (MLP)

### ⚠️ Перед импортом необходимо:

1. **Заполнить поле Gender (обязательно для MLP!):**
   - Откройте `player_list_import_ready.csv` в Excel/Numbers
   - В колонке "Gender" укажите:
     - `M` для мужчин
     - `F` для женщин
   - **Важно:** Для MLP турниров Gender обязателен!

2. **Опционально заполнить Age:**
   - Если известен возраст игроков, укажите в колонке "Age"

3. **Опционально заполнить Pool:**
   - Если нужно распределить команды по пулам, укажите номер пула (1, 2, 3...)

### Структура файла:
- ✅ First Name, Last Name - заполнены
- ✅ DUPR ID, DUPR rating - заполнены (где доступны)
- ✅ Division - заполнено (DUPR 12/14/16/18)
- ✅ Type - установлено "4v4"
- ✅ Team - заполнено
- ⚠️ Gender - **требуется заполнение**
- ⚠️ Age - опционально
- ⚠️ Pool - опционально

### Как импортировать:
1. Заполните Gender (и опционально Age/Pool)
2. Сохраните файл
3. Перейдите в турнир → Import → CSV Import
4. Загрузите файл `player_list_import_ready.csv`
5. Система автоматически создаст дивизионы, команды и игроков
