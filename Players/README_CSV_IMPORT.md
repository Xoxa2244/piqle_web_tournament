# Инструкция по созданию CSV файла для импорта игроков

## Текущий файл
- `player list.numbers` - исходный файл в формате Apple Numbers

## Требуемый формат CSV

Файл `player_list.csv` должен содержать следующие колонки:

### Обязательные колонки:
1. **First Name** - Имя игрока
2. **Last Name** - Фамилия игрока  
3. **Gender** - Пол: `M` (мужской) или `F` (женский)
4. **Age** - Возраст (число, например: 25)
5. **DUPR ID** - ID в системе DUPR (может быть пустым)
6. **DUPR rating** - Рейтинг DUPR (число, например: 3.5, может быть пустым)
7. **Division** - Название дивизиона (например: "Men's 2v2 Open")
8. **Type** - Тип игры: `1v1`, `2v2`, или `4v4`
9. **Team** - Название команды

### Опциональные колонки:
10. **Age Constraint** - Ограничение по возрасту (формат: "18-50")
11. **DUPR Constraint** - Ограничение по DUPR (формат: "2.5-4.5")
12. **Pool** - Номер пула (1, 2, 3... или пусто)

## Как создать CSV файл:

### Вариант 1: Экспорт из Numbers (рекомендуется)
1. Откройте `player list.numbers` в приложении Numbers
2. Выберите File → Export To → CSV
3. Сохраните как `player_list.csv` в папке Players
4. Откройте файл в текстовом редакторе и проверьте, что колонки соответствуют требуемому формату
5. При необходимости переименуйте колонки и добавьте недостающие

### Вариант 2: Ручное заполнение
1. Откройте `player_list.csv` (шаблон уже создан)
2. Заполните данные, скопировав из Numbers файла
3. Убедитесь, что:
   - Все обязательные колонки заполнены
   - Gender содержит только M или F
   - Age и DUPR rating - числа
   - Type соответствует: 1v1, 2v2, или 4v4

## Пример строки:
```
James,Johnson,M,25,JOHNSON001,3.2,Men's 2v2 Open,2v2,18-50,2.5-4.5,1,Team A1
```

## Важные замечания:
- Для MLP турниров Gender обязателен (M или F)
- Если Pool не указан, команда будет добавлена в первый пул
- Type определяет teamKind: 1v1 → SINGLES_1v1, 2v2 → DOUBLES_2v2, 4v4 → SQUAD_4v4
- Первый игрок в команде автоматически становится CAPTAIN
